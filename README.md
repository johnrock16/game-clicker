# game-clicker

A simple clicker game using next.js.

## Install and run
It's really simple install and run


### Install
```bash
npm install
```

### Start
```bash
npm run dev
```

## Game
You only need click and wait for money. After you receive some money you only need spend this money buying more clickers to increase your money.
